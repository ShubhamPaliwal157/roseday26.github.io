## Getting Started

### Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the page.

### Build

```bash
npm run build
```

The static files will be generated in the `out` directory.

## Deployment

This project is configured for GitHub Pages deployment. The GitHub Actions workflow will automatically build and deploy the site when changes are pushed to the `main` branch.

## Customization

- Edit messages in `src/app/page.tsx`
- Adjust colors in `src/app/globals.css`
- Modify animation parameters in component files
- Change petal/heart counts in component props
